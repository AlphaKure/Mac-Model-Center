# Mac Model Center

## Install 

1. Clone project
```
git clone https://github.com/AlphaKure/Mac-Model-Center.git
```

2. Create enviroment
```
uv venv -p python3.12
```

3. Run Code
```
python main.py
# or
uvicorn main:app --host <HOST> --port <port>
```
